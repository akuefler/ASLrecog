rawIntraface.csv contain the frame-by-frame values generated by IntraFace.  This is the file from which the training features are extracted (and/or from which they are derived as variants).

transposed_results.csv contain the frame-by-frame human-coded values.  This is the file from which truth is extracted.

Question marks indicate lack of data. 0s indicate no presence of feature. 1s indicate presence of feature.
